My Skill

Saya menguasai beberapa bahasa pemprograman yaitu : HTML, CSS, C#. untuk aplikasi yang saya kuasai yaitu : Adobe Photoshop, Adobe Ilustrator, Figma. itulah skill yang saya kuasai.

HTML

HTML adalah bahasa yang digunakan untuk membuat halaman web. Bagi yang berkecimpung di dunia perangkat gadget dan ilmu komputer , pasti sudah tidak asing lagi dengan berbagai kode di dalamnya.

CSS

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

C Sharp(C#)

C# adalah pemrograman serbaguna untuk tujuan umum untuk pemrograman server-side, aplikasi desktop/mobile, game, dll. C# juga berorientasi objek dengan konsep inheritance, class, polymorphism, dan encapsulation.

Adobe Photoshop

Adobe Photoshop atau Photoshop merupakan perangkat lunak editor citra buatan Adobe Systems yang dikhususkan untuk pengeditan foto atau gambar dan pembuatan efek, photoshop juga adalah produk terbaik dari Adobe Systems.

Adobe Ilustrator

Adobe Illustrator adalah aplikasi perangkat lunak cukup populer di kalangan profesional, desain bisnis, dan dunia seni yang digunakan oleh desainer grafis untuk membuat grafik vektor.

Figma

Figma adalah editor grafis vektor dan alat prototyping dengan berbasis web serta fitur offline tambahan yang diaktifkan oleh aplikasi desktop untuk Mac OS dan Windows.
